# 🌍 Pro Pitch Assistant

## Overview
**Pro Pitch Assistant** is an AI-powered web application designed to help Japanese freelancers and professionals win global contracts. It seamlessly transforms Japanese skill descriptions and project experiences into highly professional, culturally nuanced English cover letters and pitches.

**🔗 [Live Demo: Try the App Here!](https://pro-pitch-assistant.streamlit.app/)**

## Why I Built This
Language barriers often prevent highly skilled Japanese professionals from accessing global opportunities. Machine translation can sound robotic and miss the subtle nuances required for business proposals. I built this tool to bridge that gap, allowing users to generate tailored, persuasive English pitches in seconds.

## Key Features
* **AI-Powered Generation:** Utilizes the Google Gemini API to craft context-aware, natural-sounding English.
* **Customizable Tones:** Users can select the tone of the pitch (e.g., Professional, Friendly, Passionate) to match the specific client and job description.
* **User-Friendly Interface:** A clean, intuitive dashboard built with Streamlit, requiring no technical knowledge to use.

## Tech Stack
* **Language:** Python 3
* **Framework:** Streamlit
* **AI/LLM:** Google Gemini 1.5 Flash API
* **Deployment:** Streamlit Community Cloud
